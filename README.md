# Motion-Detector
A Python-based real-time motion detection system using OpenCV's MOG2 background subtraction algorithm. Works with both live webcam feed and pre-recorded video files.


https://github.com/user-attachments/assets/e07109b0-8fea-418a-9b1f-71ca034069c0

~ Real-time motion detection via webcam

~ Video file support (.mp4, .avi, etc.)

~ MOG2 background subtraction algorithm

~ Gaussian blur for noise reduction

~ Shadow removal using thresholding

~ Morphological operations (erode + dilate) for clean detection

~ Minimum area filter — small movements ignore 

~ Live mask window for debugging

~ Status bar — "MOTION DETECTED" / "No Motion"
